## Getting Started

First, install dependencies:

```bash
yarn 
yarn add sass
```

If you get the error

**TypeError: (...) is not a function**


```
yarn add react@16.13 react-dom@16.13
```

Open [http://localhost:3001](http://localhost:3001) with your browser.
